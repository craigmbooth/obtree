Code Examples

1. Simple Manual Table:

const columns = [
    { field: 'name', label: 'Name', visible: true, sortable: true, formatter: 'plaintext' },
    { field: 'email', label: 'Email', visible: true, sortable: true, formatter: 'email' }
];

const data = [
    { name: 'John Doe', email: 'john@example.com' },
    { name: 'Jane Smith', email: 'jane@example.com' }
];

const table = buildTableManual('#my-table', data, columns);

2. Table from Backend Configuration:

// Fetch table config and data from backend
const table = await buildTableFromConfig(
    '#users-table',      // Container selector
    '/api/users',        // Data endpoint
    'User'               // Model name
);

3. Adding Backend Config to Models:

class MyModel(Base, TableConfigMixin):
    __table_config__ = {
        'columns': [
            {
                'field': 'id',
                'label': 'ID',
                'visible': False,  # Hidden column
                'sortable': True,
                'formatter': 'plaintext'
            },
            {
                'field': 'email',
                'label': 'Email',
                'visible': True,
                'sortable': True,
                'width': 250,
                'formatter': 'email'
            }
        ],
        'default_sort': {'field': 'created_at', 'dir': 'desc'}
    }
